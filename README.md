# 🌿 Проект «Зелёный Офис»

Комплексная информационная кампания по внедрению раздельного сбора отходов в офисе.

## 📋 О проекте

**Цель:** Сократить объём мусора, отправляемого на полигоны, и сформировать у сотрудников экологическую культуру.

**Задача:** Создать полноценную внутреннюю коммуникационную кампанию с помощью AI-инструментов — от текста и визуала до видео и интерактивного чат-бота.

**Результат:** Готовые к использованию материалы для запуска экологической инициативы в компании.

## 🛠 Использованные инструменты

- **GigaChat** — тексты, сценарии, база знаний для чат-бота
- **Шедеврум** — плакаты, иконки для урн
- **HeyGen** — видеоинструктаж с AI-аватаром
- **Suno** — музыка для видео
- **Aimylogic** — платформа для чат-бота

## 📁 Структура проекта

- `01-documentation/` — Документация и ТЗ
- `02-text-and-scenario/` — Тексты и сценарии
- `03-visual-materials/` — Визуальные материалы
- `04-video/` — Видеоматериалы
- `05-chatbot/` — Чат-бот «Эко-помощник»
- `06-results/` — Итоги проекта

## 🎯 Компоненты кампании

### 1️⃣ Памятка для сотрудников
Инструкция по правильной сортировке отходов с указанием цветов контейнеров и типов отходов.

### 2️⃣ Видеоинструктаж (30 секунд)
Короткое видео с AI-аватаром, объясняющее правила раздельного сбора отходов.

### 3️⃣ Визуальные материалы
- Плакат «Сортируй — помогай планете!»
- Иконки для контейнеров (бумага, пластик, смешанные отходы)

### 4️⃣ Чат-бот «Эко-помощник»
Интерактивный помощник, который отвечает на вопросы сотрудников о правилах сортировки отходов.

## 💡 Особенности проекта

- **Полный цикл:** от идеи до готовых материалов
- **AI-first подход:** все материалы созданы с помощью нейросетей
- **Быстрый запуск:** кампания готова к немедленному использованию
- **Масштабируемость:** материалы можно адаптировать под любую компанию

## 📊 Результат

Проект демонстрирует, как AI-специалист может в одиночку и за минимальное время запустить полноценную внутреннюю коммуникационную кампанию. Все материалы объединены одной целью и готовы к использованию.

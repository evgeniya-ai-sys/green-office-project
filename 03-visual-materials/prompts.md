# 🎨 Промпты для визуальных материалов

## 📌 Плакат «Сортируй — помогай планете!»

**Промпт:**
Плакат для офиса, три урны (синяя, жёлтая, серая), офисные сотрудники сортируют отходы, современная плоская иллюстрация, зелёные и голубые тона, вертикальная ориентация, 2:3.

**Назначение:** Информационный плакат для размещения в офисе рядом с контейнерами для раздельного сбора отходов.

---

## 🗑️ Иконки для урн

**Промпт:**
Три иконки — газета, бутылка, яблочный огрызок. Стиль — минимализм, line art, чёрно-белые, квадратный формат, 1:1.

**Назначение:** Иконки для размещения на контейнерах (бумага, пластик, смешанные отходы).

---

## 💡 Советы по генерации

- Используйте вертикальную ориентацию для плакатов (2:3)
- Для иконок выбирайте квадратный формат (1:1)
- Минималистичный стиль лучше воспринимается в офисе
- Зелёные и голубые тона поддерживают экологическую тематику
